# Clear Browser Cache Instructions

## Chrome/Edge:
1. Open DevTools (F12)
2. Right-click refresh button → "Empty Cache and Hard Reload"
3. Or: Ctrl+Shift+R (Windows) / Cmd+Shift+R (Mac)

## Firefox:
1. Open DevTools (F12)
2. Settings → Network → "Disable cache"
3. Or: Ctrl+F5 (Windows) / Cmd+Shift+R (Mac)

## Alternative:
- Open in Incognito/Private mode
- Or clear all browser data from settings
